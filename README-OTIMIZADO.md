# Cardápio Online (React + Firebase + Tailwind)

## Como rodar
1. `cp .env.example .env.local` e preencha suas chaves do Firebase.
2. `npm i`
3. `npm run dev`

## Recursos
- Login com Google (Firebase Auth)
- Firestore (pedidos)
- Avatar com foto do Google e fallback de inicial
- Navbar fixa e legível no mobile
- Cards de restaurantes clicáveis
- Feedback ao adicionar ao carrinho (animação + toast)
- Filtro por tipo de restaurante
